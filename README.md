# Carbon-24 Clustering Dashboard

Dashboard trực quan hóa kết quả phân nhóm các dạng thù hình Carbon dựa trên đặc trưng hình học ô đơn vị và mức ổn định năng lượng.

## Features

- PCA 2D/3D visualization
- Filter by clustering algorithm
- Filter by cluster
- Energy distribution by cluster
- Density vs energy analysis
- Cluster profile table
- Interactive data table

## Dataset

The dashboard uses processed Carbon-24 clustering data stored in:

```text
data/all_models_dashboard.csv
```
